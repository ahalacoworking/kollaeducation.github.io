    Indonesia's foreign domestic workers have a way to pick up coding skills, thanks to new courses being organised in the cities they work in. The two-month-long coding course, is being run by the Indonesian government's Creative Economy Agency (Bekraf), and is similar to one it's already started running back home for stay-at-home mothers, called "Coding Mum."

    What our People Say

    Sebelum mengikuti Coding Mum saya adalah IRT biasa yang tidak tahu mengenai programming. Setelah mengikuti Coding Mum, saya banyak belajar dan dibantu untuk mengerti programming khususnya di-frontend development dimana diakhir kelas diajarkan untuk membuat website sendiri.

    Setelah mengikuti Coding Mum saya berkesempatan untuk mengikuti project dari Garuda, yaitu pembuatan kios digital. Selain itu saya juga membuat website sendiri yaitu catatancoding.com. Saya ingin berterimakasih kepada Coding Mum karena telah memberikan saya kesempatan untuk belajar mengenai dunia pemograman lebih lanjut.
