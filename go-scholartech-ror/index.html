GO-SCHOLAR TECH RUBY ON RAILS

Apa itu GO-SCHOLAR TECH Ruby on Rails?

GO-JEK bekerja sama dengan KollaEdu, mengajak talenta talenta terbaik Indonesia untuk mengikuti GO-SCHOLAR TECH Ruby on Rails. Program ini memberikan beasiswa pelatihan programming intensif untuk meningkatkan kemampuan programming kamu ke level yang lebih tinggi!

Program beasiswa pelatihan Ruby on Rails intensif ini menerapkan filosofi Trifecta of Learning: belajar di kelas, mendapatkan mentoring, dan mengerjakan proyek riil. Selama program, peserta akan mendapatkan kesempatan untuk:

  • Mendapatkan pelatihan intensif bahasa pemrograman Ruby on Rails dan penerapannya di industri, serta pengembangan soft skill dipandu para pengajar berkualitas
  • Mendapatkan masukan melalui mentoring langsung dari engineer senior di GO-JEK
  • Mengaplikasikan teori yang didapat dengan mengerjakan proyek rill dalam kegiatan pemagangan di GO-JEK

Mengapa GO-JEK dan KollaEdu berkolaborasi?

Jadikan dirimu programmer yang siap bersaing di tingkat internasional!

A. Industri teknologi di Indonesia mengalami pertumbuhan pesat selama beberapa tahun terakhir. Hal ini dikarenakan banyaknya solusi yang dihasilkan oleh perusahaan teknologi untuk menyelesaikan masalah dengan cara yang cepat dan inovatif.

B. Meskipun demikian, masih banyak permasalahan yang saat ini belum dapat ditemukan solusinya. Maka dari itu, dibutuhkan talenta talenta kelas dunia di bidang teknologi untuk dapat menciptakan solusi solusi yang belum pernah terpikirkan sebelumnya.

C. Dengan jumlah penduduk yang mencapai 250 juta orang, Indonesia memiliki potensi sumber daya manusia yang sangat luar biasa. Jumlah putra-putri Indonesia yang berprestasi di bidang teknologi pada taraf internasional juga meningkat setiap tahunnya. Sayangnya masih banyak potensi yang belum dapat dimaksimalkan karena kurangnya wadah untuk meningkatkan kompetensi di bidang teknologi.

D. GO-JEK melalui GO-JEK Academy (GO-Academy) dan KollaEdu bekerjasama untuk memberikan kesempatan meningkatkan kemampuan programming kamu ke level yang lebih tinggi lewat program GO-SCHOLAR TECH Ruby on Rails.

Lembaga Edukasi Literasi Teknologi

KollaEdu adalah lembaga edukasi informal yang bervisi menumbuhkembangkan kemampuan literasi baca tulis kode bagi masyarakat luas, agar mengerti bagaimana teknologi bekerja dan mampu menjadi pencipta teknologi.

Dengan benang merah berupa semangat menyebarkan literasi kode sehingga menciptakan masyarakat yang tak hanya mampu menggunakan teknologi tapi juga mampu menciptakan teknologi, GO-Academy dan KollaEdu berkerjasama dalam program GO-SCHOLAR TECH Ruby on Rails.

Lulusan KollaEdu berasal dari berbagai latar belakang, mulai dari remaja, ibu rumah tangga, pekerja migran hingga para programmer dengan kemampuan mumpuni. KollaEdu percaya bahwa teknologi dapat menjadi salah satu solusi bagi banyak masalah yang dihadapi oleh masyarakat di negeri ini.

Pertanyaan Umum


1. Apakah saya harus memiliki latar belakang programming untuk ikut kegiatan ini?

Tentu saja.

Program ini ditujukan bagi para programmer yang menguasai setidaknya satu bahasa pemrograman dengan basis OOP kuat: Ruby, Java, C# atau Phyton serta memiliki pemahaman tentang Command-Line Interface.

2. Apa aja tahapan seleksi yang harus saya lewati untuk mengikuti program ini?

Seleksi terdiri dari 3 tahapan yaitu:

  1. 1. Seleksi Online Testing Coding
  2. 2. Seleksi Onsite Testing Coding
  3. 3. Seleksi Wawancara

3. Berapa orang yang akan diterima untuk mengikuti program ini?

Ada deh. Yang penting kamu coba saja dulu.

4. Bagaimana rincian pelaksanaan program?

Total program akan berlangsung selama 6 bulan dan dibagi dalam 2 tahapan.

Pada tahap pertama akan diadakan program pelatihan selama tiga bulan. Setiap bulannya akan diadakan evaluasi untuk mengukur pemahaman peserta. Materi yang akan kamu dapatkan berupa materi tentang backend programming dengan framework Ruby on Rails.

Materi mencakup penggunakan teknologi Ruby on Rails untuk membuat backend yang reliable dan scalable in mind, termasuk materi Test Driven Development dalam pengembangan aplikasinya.

Pada tahap kedua kamu akan mengikuti pemaganan di GO-JEK selama 3 bulan. Kamu akan dibekali juga dengan berbagai pelatihan soft skill.

5. Di mana kelas akan dilaksanakan?

Kelas akan dilaksanakan di Kolla Space Kedoya, Jakarta Barat.

Pemagangan akan dilakukan di kantor GO-JEK, Blok M, Jakarta Selatan.

Peserta terpilih tidak dapat mengikuti program secara remote sehingga disarankan untuk tinggal di Jakarta selama program berlangsung.

6. Apakah saya masih bisa melakukan kegiatan lain seperti bekerja atau berkuliah di luar jadwal kelas?

Program akan berlangsung dari pukul 09.00-17.00 dari hari Senin hingga Jumat.

Di luar kelas, peserta juga akan diberikan tugas.

Karena itu, kami tidak menyarankan kamu mengikuti kegiatan ini sembari bekerja atau berkuliah.

7. Apakah saya diperbolehkan untuk izin di tengah program ini?

Setiap harinya kami akan mengajarkan hal baru kepada kamu sehingga alangkah baiknya jika kamu tidak melewatkan materi yang kami berikan.

Kamu diperbolehkan izin jika memang ada hal yang benar benar mendesak seperti sakit, ada anggota keluarga yang meninggal, bencana alam, dll.

8. Berapa biaya yang diperlukan untuk mengikuti program GO-SCHOLAR TECH Ruby on Rails?

Untuk mengikuti program ini kamu hanya perlu melakukan investasi sebesar 36 juta rupiah.

Tapi jangan khawatir, bagi para peserta yang lolos seleksi dengan kriteria memuaskan akan mendapar beasiswa dari GO-JEK, sehingga bisa mengikuti program tanpa harus membayar alias GRATIS.

Kamu hanya perlu membayar uang jaminan sebesar 5 juta rupiah yang akan dikembalikan begitu kamu menyelesaikan program.

9. Apakah saya harus bekerja di GO-JEK setelah lulus dari program ini?

Jadi begini, selama program ini kami akan menilai kecocokan kamu dengan GO-JEK, begitu pula sebaliknya.

Jika di akhir program baik kamu maupun GO-JEK menemukan kecocokan, ada kemungkinan kamu akan melanjutkan karir di GO-JEK.

Adapun selama program kamu tidak diperbolehkan untuk mendaftar pekerjaan di tempat lain.
